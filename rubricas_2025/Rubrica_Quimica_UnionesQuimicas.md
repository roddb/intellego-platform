## EJERCICIO TIPO 1: Identificación de Tipo de Enlace

**Contexto típico:** Dados dos compuestos, identificar el tipo de enlace, justificar con electronegatividad y predecir propiedades.

| FASE | EXCELENTE (12.5-11 pts) | BUENO (10-8 pts) | REGULAR (7-5 pts) | INSUFICIENTE (4-0 pts) |
|------|-------------------------|------------------|-------------------|------------------------|
| **F1: Comprensión del Problema** | Identifica los compuestos y sus elementos constituyentes. Reconoce metal/no metal. Comprende que debe determinar tipo de enlace y justificar con EN. | Identifica compuestos y elementos. Comprende la tarea general. | Datos parciales. No clasifica elementos como metal/no metal. | No identifica datos o no comprende el problema. |
| **F2: Identificación de Variables** | Elementos de cada compuesto. Clasificación: metal/no metal. Valores de EN de cada elemento (tabla dada). ΔEN = |EN₁ - EN₂|. | Variables correctas. Identifica EN de elementos. | Algunas variables. Error en clasificación de elementos. | Variables incorrectas o ausentes. |
| **F3: Selección de Herramientas** | Criterios: Metal + No metal → iónico, No metal + No metal → covalente. ΔEN > 1.7 → iónico, ΔEN < 1.7 → covalente. Propiedades asociadas a cada tipo. | Criterios correctos y completos. | Criterios parciales o incompletos. | Criterios incorrectos. |
| **F4: Estrategia y Ejecución** | Calcula ΔEN. Clasifica enlace. Justifica con ambos criterios (elementos y ΔEN). Predice propiedades correctamente. | Clasificación correcta con justificación adecuada. | Clasificación correcta pero justificación incompleta. | Clasificación incorrecta o sin justificación. |

### Ejemplo de resolución esperada:

```
Compuestos: NaCl y H₂O
Datos de EN: Na = 0.9, Cl = 3.0, H = 2.1, O = 3.5

a) Tipo de enlace:

NaCl:
- Na: metal alcalino + Cl: halógeno (no metal) → Metal + No metal
- ΔEN = |3.0 - 0.9| = 2.1 > 1.7
- Conclusión: ENLACE IÓNICO

H₂O:
- H: no metal + O: no metal → No metal + No metal
- ΔEN = |3.5 - 2.1| = 1.4 < 1.7
- Conclusión: ENLACE COVALENTE (polar)

b) Justificación con electronegatividad:
- NaCl: Gran diferencia de EN (2.1) indica transferencia de electrones
- H₂O: Menor diferencia de EN (1.4) indica compartición de electrones

c) Predicción de punto de fusión:
- NaCl (iónico): Alto punto de fusión (~801°C) debido a fuertes 
  atracciones electrostáticas entre iones
- H₂O (covalente): Bajo punto de fusión (0°C) debido a fuerzas 
  intermoleculares más débiles entre moléculas
```

### Ejemplo de resolución esperada:

```
Tabla de propiedades:
| Sustancia | P. fusión | Conduce elec. | Soluble |
|-----------|-----------|---------------|---------|
| X         | 801°C     | Sí (fundido)  | Sí      |
| Y         | -78°C     | No            | Poco    |

Afirmación: "X tiene enlace iónico e Y tiene enlace covalente"

Análisis de X:
✓ Punto de fusión alto (801°C) → característico de iónico
✓ Conduce electricidad fundido → iones móviles (iónico)
✓ Soluble en agua → compuesto iónico polar
→ Conclusión: X tiene ENLACE IÓNICO

Análisis de Y:
✓ Punto de fusión muy bajo (-78°C) → característico de covalente
✓ No conduce electricidad → no hay iones ni electrones libres
✓ Poco soluble → molécula no polar o poco polar
→ Conclusión: Y tiene ENLACE COVALENTE

Conclusión: VERDADERO
La afirmación es correcta. Las propiedades de X corresponden a un 
compuesto iónico (probablemente NaCl) y las de Y a un compuesto 
covalente molecular (probablemente CO₂, que sublima a -78°C).
```

## TIPOS DE ENLACE Y SUS PROPIEDADES

| Propiedad | IÓNICO | COVALENTE | METÁLICO |
|-----------|--------|-----------|----------|
| **Elementos** | Metal + No metal | No metal + No metal | Metal + Metal |
| **ΔEN** | > 1.7 | < 1.7 | ≈ 0 |
| **Punto de fusión** | Alto (>400°C) | Bajo (<300°C) | Variable (alto) |
| **Conductividad sólido** | No | No | Sí |
| **Conductividad fundido** | Sí | No | Sí |
| **Conductividad en solución** | Sí (si soluble) | No | N/A |
| **Solubilidad en agua** | Generalmente sí | Variable | No |
| **Estructura** | Red cristalina | Moléculas | Red metálica |

## CRITERIO DE CLASIFICACIÓN POR ΔEN

| ΔEN | Tipo de enlace | Característica |
|-----|----------------|----------------|
| 0 - 0.4 | Covalente no polar | Compartición igual |
| 0.4 - 1.7 | Covalente polar | Compartición desigual |
| > 1.7 | Iónico | Transferencia de e⁻ |
