## EJERCICIO TIPO 1: Cálculos Atómicos y Configuración Electrónica

**Contexto típico:** Dado un átomo neutro con ciertos protones y número másico, calcular neutrones, electrones y escribir la configuración electrónica.

| FASE | EXCELENTE (12.5-11 pts) | BUENO (10-8 pts) | REGULAR (7-5 pts) | INSUFICIENTE (4-0 pts) |
|------|-------------------------|------------------|-------------------|------------------------|
| **F1: Comprensión del Problema** | Identifica: Z (protones), A (número másico), átomo neutro (e⁻ = p⁺). Comprende que debe calcular n y escribir configuración electrónica completa. | Identifica datos principales. Comprende las relaciones básicas. | Datos parciales. Confunde Z con A o no reconoce átomo neutro. | No identifica datos o no comprende la estructura atómica. |
| **F2: Identificación de Variables** | Z = p⁺ = 26, A = 56, e⁻ = 26 (neutro), n = A - Z. Identifica el elemento (Fe). | Variables correctas con identificación del elemento. | Algunas variables. No identifica el elemento o error en n. | Variables incorrectas o ausentes. |
| **F3: Selección de Herramientas** | n = A - Z, e⁻ = Z (neutro). Regla de Aufbau: 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹⁰ 4p⁶... Capacidad de cada subnivel. | Fórmulas correctas y secuencia de Aufbau completa. | Fórmulas correctas pero Aufbau incompleto o con errores. | Fórmulas incorrectas o sin Aufbau. |
| **F4: Estrategia y Ejecución** | Calcula n = 56 - 26 = 30. e⁻ = 26. Escribe configuración correcta distribuyendo 26 e⁻. | Procedimiento correcto con error menor en configuración. | n y e⁻ correctos pero configuración con errores. | Cálculos erróneos o configuración muy incorrecta. |

### Ejemplo de resolución esperada:

```
Datos: Z = 26 protones, A = 56, átomo neutro

a) Número de neutrones:
   n = A - Z = 56 - 26 = 30 neutrones

b) Número de electrones:
   Átomo neutro → e⁻ = p⁺ = Z = 26 electrones

c) Configuración electrónica (26 e⁻):
   1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d⁶
   (2 + 2 + 6 + 2 + 6 + 2 + 6 = 26 ✓)

   Elemento: Hierro (Fe)
```

## EJERCICIO TIPO 2: Isótopos y Clasificación de Especies

**Contexto típico:** Dadas dos especies químicas con sus partículas, determinar si son isótopos, isóbaros o isótonos.

| FASE | EXCELENTE (12.5-11 pts) | BUENO (10-8 pts) | REGULAR (7-5 pts) | INSUFICIENTE (4-0 pts) |
|------|-------------------------|------------------|-------------------|------------------------|
| **F1: Comprensión del Problema** | Identifica p⁺, e⁻, n de ambas especies. Comprende las definiciones: isótopos (mismo Z), isóbaros (mismo A), isótonos (mismo n). | Identifica datos de ambas especies. Conoce las definiciones básicas. | Datos parciales. Confunde definiciones. | No identifica datos o no conoce las definiciones. |
| **F2: Identificación de Variables** | Especie X: Z_X, A_X, n_X. Especie Y: Z_Y, A_Y, n_Y. Calcula A = Z + n para cada una. | Variables correctas para ambas especies. | Algunas variables. Error en cálculo de A. | Variables incorrectas o incompletas. |
| **F3: Selección de Herramientas** | Criterios: Z igual → isótopos, A igual → isóbaros, n igual → isótonos. Identifica elemento por Z en tabla periódica. | Criterios correctos con aplicación adecuada. | Criterios parciales o mal aplicados. | Criterios incorrectos. |
| **F4: Estrategia y Ejecución** | Compara Z, A, n. Identifica correctamente la relación. Nombra el elemento. Justifica V o F con argumentos claros. | Identificación correcta con justificación menor. | Identificación correcta pero justificación pobre. | Identificación incorrecta o sin justificación. |

### Ejemplo de resolución esperada:

```
Especie X: 18 p⁺, 18 e⁻, 22 n → A_X = 18 + 22 = 40
Especie Y: 18 p⁺, 18 e⁻, 20 n → A_Y = 18 + 20 = 38

Análisis:
- Z_X = Z_Y = 18 (mismo número atómico) ✓
- A_X = 40 ≠ A_Y = 38 (diferente número másico) ✓
- n_X = 22 ≠ n_Y = 20 (diferente número de neutrones) ✓

Elemento: Z = 18 → Argón (Ar)

Conclusión: VERDADERO
Las especies X e Y son ISÓTOPOS del Argón porque:
- Tienen el mismo número de protones (Z = 18)
- Tienen diferente número de neutrones (22 y 20)
- Por lo tanto, diferente número másico (40 y 38)

Se representan como: ⁴⁰Ar y ³⁸Ar
```

## CONCEPTOS CLAVE

| Concepto | Definición | Fórmula |
|----------|------------|---------|
| Número atómico (Z) | Cantidad de protones | Z = p⁺ |
| Número másico (A) | Protones + Neutrones | A = Z + n |
| Átomo neutro | e⁻ = p⁺ | e⁻ = Z |
| Ion positivo (catión) | e⁻ < p⁺ | e⁻ = Z - carga |
| Ion negativo (anión) | e⁻ > p⁺ | e⁻ = Z + carga |

## PRINCIPIO DE AUFBAU - ORDEN DE LLENADO

```
1s² → 2s² → 2p⁶ → 3s² → 3p⁶ → 4s² → 3d¹⁰ → 4p⁶ → 5s² → 4d¹⁰ → 5p⁶ → 6s² → 4f¹⁴ → 5d¹⁰ → 6p⁶ → 7s²...
```

| Subnivel | Capacidad máxima |
|----------|------------------|
| s | 2 e⁻ |
| p | 6 e⁻ |
| d | 10 e⁻ |
| f | 14 e⁻ |
