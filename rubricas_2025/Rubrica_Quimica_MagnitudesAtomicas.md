## EJERCICIO TIPO 1: Conversiones Masa-Moles-Partículas

**Contexto típico:** Dada una masa de sustancia, calcular moles, moléculas y átomos totales.

| FASE | EXCELENTE (12.5-11 pts) | BUENO (10-8 pts) | REGULAR (7-5 pts) | INSUFICIENTE (4-0 pts) |
|------|-------------------------|------------------|-------------------|------------------------|
| **F1: Comprensión del Problema** | Identifica: masa (g), masa molar (g/mol), Nₐ. Comprende la cadena: g → mol → moléculas → átomos. Reconoce cuántos átomos hay por molécula. | Identifica datos principales. Comprende las conversiones básicas. | Datos parciales. No comprende la relación moléculas-átomos. | No identifica datos o no comprende las conversiones. |
| **F2: Identificación de Variables** | m (g), M (g/mol), n (mol), N (moléculas), N_átomos. Para CO₂: 3 átomos/molécula (1C + 2O). | Variables correctas. Identifica átomos por molécula. | Algunas variables. Error en átomos por molécula. | Variables incorrectas o ausentes. |
| **F3: Selección de Herramientas** | n = m/M, N = n × Nₐ, N_átomos = N × (átomos/molécula). Secuencia completa. | Fórmulas correctas con alguna omisión. | Algunas fórmulas correctas. | Fórmulas incorrectas. |
| **F4: Estrategia y Ejecución** | Calcula n, N, N_átomos correctamente. Usa notación científica apropiada. Unidades correctas. | Procedimiento correcto con error de cálculo menor. | Procedimiento reconocible con errores. | Sin procedimiento o cálculos erróneos. |

### Ejemplo de resolución esperada:

```
Datos: m = 44 g de CO₂, M = 44 g/mol, Nₐ = 6.02 × 10²³ mol⁻¹

a) Moles de CO₂:
   n = m/M = 44 g / 44 g/mol = 1 mol

b) Número de moléculas:
   N = n × Nₐ = 1 mol × 6.02 × 10²³ mol⁻¹ = 6.02 × 10²³ moléculas

c) Número total de átomos:
   CO₂ tiene 1 átomo de C + 2 átomos de O = 3 átomos/molécula
   N_átomos = 6.02 × 10²³ × 3 = 1.806 × 10²⁴ átomos
```

## EJERCICIO TIPO 2: Comparación de Muestras

**Contexto típico:** Comparar dos muestras de diferentes sustancias para verificar si tienen el mismo número de moles/moléculas.

| FASE | EXCELENTE (12.5-11 pts) | BUENO (10-8 pts) | REGULAR (7-5 pts) | INSUFICIENTE (4-0 pts) |
|------|-------------------------|------------------|-------------------|------------------------|
| **F1: Comprensión del Problema** | Identifica masas y masas molares de ambas muestras. Comprende que igual número de moles = igual número de moléculas. | Identifica datos de ambas muestras. Comprende la relación mol-moléculas. | Datos parciales. Comprensión superficial. | No identifica datos o no comprende la comparación. |
| **F2: Identificación de Variables** | Muestra A: m_A, M_A, n_A. Muestra B: m_B, M_B, n_B. Comparar n_A vs n_B. | Variables correctas para ambas muestras. | Algunas variables correctas. | Variables incorrectas. |
| **F3: Selección de Herramientas** | n = m/M para cada muestra. Criterio: n_A = n_B → mismo número de moléculas. | Fórmula correcta con criterio de comparación. | Fórmula correcta pero criterio incompleto. | Fórmula incorrecta. |
| **F4: Estrategia y Ejecución** | Calcula n para ambas muestras. Compara valores. Concluye V o F con justificación clara. | Cálculos correctos con conclusión adecuada. | Cálculos con errores pero conclusión coherente. | Cálculos erróneos o conclusión sin justificación. |

### Ejemplo de resolución esperada:

```
Muestra A: 32 g de O₂, M = 32 g/mol
Muestra B: 18 g de H₂O, M = 18 g/mol

Cálculos:
n_A = m_A/M_A = 32/32 = 1 mol de O₂
n_B = m_B/M_B = 18/18 = 1 mol de H₂O

Comparación: n_A = n_B = 1 mol

Moléculas:
N_A = 1 × 6.02 × 10²³ = 6.02 × 10²³ moléculas de O₂
N_B = 1 × 6.02 × 10²³ = 6.02 × 10²³ moléculas de H₂O

Conclusión: VERDADERO
Ambas muestras contienen el mismo número de moléculas (6.02 × 10²³)
porque cada una contiene exactamente 1 mol de sustancia.
```

## FÓRMULAS CLAVE

| Concepto | Fórmula | Unidades |
|----------|---------|----------|
| Moles | n = m/M | mol |
| Masa | m = n × M | g |
| Partículas | N = n × Nₐ | moléculas o átomos |
| Moles desde partículas | n = N/Nₐ | mol |

---

## CONSTANTE DE AVOGADRO

```
Nₐ = 6.02 × 10²³ mol⁻¹
```

**Significado:** 1 mol de cualquier sustancia contiene 6.02 × 10²³ partículas (átomos, moléculas, iones, etc.)

## CADENA DE CONVERSIONES

```
           ÷M                    ×Nₐ
MASA (g) ────────→ MOLES (mol) ────────→ MOLÉCULAS
           ×M                    ÷Nₐ
         ←────────             ←────────

                                          ×(átomos/molécula)
MOLÉCULAS ──────────────────────────────────────────────────→ ÁTOMOS
                                          ÷(átomos/molécula)
          ←──────────────────────────────────────────────────
```
